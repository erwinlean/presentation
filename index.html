Landing page Caliz espacio integral

La landing page de Caliz Espacio Integral ofrece una experiencia integral de bienestar, con clases de yoga, masajes y Reiki para equilibrar cuerpo y mente. Con un diseño atractivo y una navegación sencilla, atrae a los visitantes hacia la relajación y la sanación.

Proyecto Life

Aplicación tiene como objetivo ofrecer un sistema de inicio de sesión de usuario, donde se almacenan los datos del usuario y se calcula el IMC (Índice de Masa Corporal), el peso ideal y las calorías quemadas diariamente según su nivel de actividad física. Además, proporciona gráficos informativos para el seguimiento del progreso del usuario.

Sistema de Gestión de Contenidos (CMS) PIM

Portal para la carga de productos de proveedores, vía API hacia un PIM (Gestor de Información de Productos). Incluye la gestión de usuarios, la creación de productos y la descarga de productos en formato PDF, entre otras funciones de administración requeridas por el cliente.

Tizzina barbershop manejo de cuentas

La aplicación de contabilidad y gestión de cuentas de Tizzianos Barbershop se utiliza para llevar un registro de los ingresos y gastos diarios, semanales y mensuales, además de administrar los precios en la plataforma web del negocio. Esto es fundamental para una gestión financiera precisa y eficiente.
